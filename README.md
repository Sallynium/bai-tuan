# 🐶 桌面小狗互動介面（bai-tuan）

- **這是什麼？** 一個療癒系「養小狗」網頁：卡通狗狗會自己走動，可以摸牠、餵牠、搓肚子，冷落牠會委屈。單一 `index.html`，雙擊即開。
- **給誰用？** 自用（療癒 + 前端練習）。
- **MVP（v1 完成 = 全勾）**
  - [ ] 場景 + 狗狗靜態 SVG（伯恩山），idle 動態（走動/坐下/眨眼/搖尾）
  - [ ] 撫摸 + 愛心粒子 + happy；餵食完整流程（含按鈕冷卻）
  - [ ] 連點翻肚、搓肚踢腿
  - [ ] mood 系統 + lonely / comforted / sleeping
  - [ ] 薩摩耶切換、對話泡泡、雲朵、reduced-motion
- **明確不做什麼？**
  - 不存檔（無 localStorage）、無音效、無外部資源（CDN/圖片/字型）
  - 不加規格書以外的功能或改動數值

## 本機執行

雙擊 `index.html` 即可，或任一靜態伺服器：

```
npx serve .
```

規格書：見開發對話（實作嚴格依照規格書，勿自行加料）。
